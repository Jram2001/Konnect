how macro matching works
1. user watches "tesla_company" which has macro_group_keys: ["ev_sector", "big_tech"]
2. collect all unique group_keys from user's watchlist entities
3. query connections: { is_macro: true, macro_group_keys: { $in: ["ev_sector", "big_tech"] }, discovered_at: today }
4. merge with micro connections, rank by impact_score, build digest
